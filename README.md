## 💻 Getting Started <a name="getting-started"></a>

> How to get started and make use of this project.

To get a local copy up and running, follow these steps.

### Prerequisites

In order to run this project you need:  A web browser

### Setup

Clone this [repository](https://github.com/jlvFlores/redux-toolkit-warm-up) to your desired folder

### Install

Install all dependencies to this project with: `npm install`

### Usage

To run the project, execute the following command:  `npm run dev`

### Run tests

To run tests, run the following command: `npm run dev`

### Deployment

In the future you will be able to deploy this project: Using a deploy service.
